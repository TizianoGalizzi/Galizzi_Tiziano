    • b) Para que se utilizan las siguientes etiquetas:

  • head: El tag head tiene la funcion de contener la metadata(datos sobre otros datos) del archivo html, asi como los tags: title, style, meta, link, script y base
  • title: El tag title define el titulo del documento, el cual solo puede ser texto y sera mostrado en la barra de navegador. Su uso es esencial, ya que este es uno de los factores importantes que influye al SEO(motor optimizador de busqueda) al ser buscada la pagina por los usuarios
  • meta: Es usado para definir metadata en el documento HTML. Este tag va siempre en el head.
  • style: El tag style es utilizado para dar estilo css dentro del archivo html
  • body: El body contiene todo el contenido visible de una pagina web.
  • (h1,h2,h3,h4,h5,h6): Los tags H definen los encabezados del cuerpo del body, cada uno de ellos comprenden un tamaño diferente en su contenido
  • a: Sirve para definir un hiperenlace, lo que es usado para redirigir de una pagina a otra. Este tag requiere del atributo href, que sirve para definir el link al que se quiere redirigir
  • strong: Dota a un texto con mayor importancia.
  • br: Permite realizar un salto de linea dentro de la pagina
  • ul: Define en su interior a los tags li, haciendo que estos se creen de manera desordenada, es decir, sin numeracion.
  • li: Permite enlistar texto en forma de items
  • p: En el podemos escribir parrafos
  • span: El tag span es un contenedor en linea que es usado para darle color a una porcion de texto, por lo tanto. Sirve para remarcar con color una fraccion de texto
  • table: El tag table crea un tipo de formato de tabla. Dentro de este tag contenedor, van los tags tr y td.
  • tr: El tag tr (tableRow) crea una fila dentro de la tabla.
  • td: El tag td crea la celda donde ira el contenido de la tabla
  • img: El tag img (image) crea un espacio de referencia para la imagen que se desea insertar. Este tag requiere de los atributos src(source) donde se especifica la ruta de la imagen, y alt, que es un texto alternativo en caso de que no se logre mostrar la imagen.

    c)La etiqueta script puede ir dentro de las etiquetas de de body o head, hay alguna diferencia donde pueda ir?


  • El debate entre poner scripts al final del body o en el head se debe a una cuestión de renderización de la página.
    Siempre lo mejor sera buscar que el archivo HTML no reciba bloqueos (problema que surge en la lectura de scripts durante la carga de la pagina.), pero existen veces en las que los scripts tienen una importancia critica en el funcionamiento de la pagina y han de ir en el head para que la pagina inicialize correctamente. En conclusion, si el script es de suma importancia para el buen funcionamiento de la pagina, ponerlo en el head. Sino, ubicarlo debajo o dentro del body.
